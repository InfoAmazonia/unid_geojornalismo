# Vullneráveis do clima

Os arquivos e scripts utilizados na análise de zonas de risco geológico na 
região amazônica estão organizados nesta pasta com a seguinte estrutura:

- `data-raw`: dados originais das fontes, com eventuais filtros ou cruzamentos
- `data-tidy`: dados intermediários utilizados para as análises após limpeza dos dados brutos
- `R`: scripts em R com arrumações dos dados
- `relatorio`: relatório em Quarto com os principais achados, com resultados em
  PDF, html e docx.

## Reprodução da análise

Os números dos scripts R indicam a ordem em que foram executados para no 
processo de obtenção, limpeza e análise dos dados.
